# Changelog

Все значимые изменения в этом проекте будут отражены в этом файле.

## [0.1.0] - 2026-07-16

### Добавлено (Added)
- Первый релиз протокола Undertow
- Генерация PeerId на основе X25519
- Клиент для взаимодействия с Beacon-сервером (регистрация, ретрансляция)
- TUI-клиент (`undertow-client`) для тестирования
- STUN-клиент для определения внешнего IP
- Поиск пиров в LAN
- Модуль `network`: управление соединениями, NAT-обход (stub), DHT (stub)
- Модуль `protocol`: форматы пакетов, сериализация bincode
- Модуль `storage`: профиль пользователя (ключи, настройки)
- Поддержка Linux и Termux (Android)
- Документация на русском и английском языках

### Изменено (Changed)
- Нет (первый релиз)

### Исправлено (Fixed)
- Нет (первый релиз)

### Безопасность (Security)
- Используется X25519 для генерации ключей
- SHA-256 для создания PeerId
